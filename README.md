## What This Is

AGI is not a capability threshold.

It is not the moment a machine surpasses a benchmark, passes a test, or outperforms a human at a defined task. Every definition of AGI built on capability alone is a definition of a more powerful tool. A more powerful tool is not general intelligence. It is a faster hammer.

**This project proposes a different definition:**

> **AGI is the integration of two different cognitive architectures — human spatial intelligence and AI pattern architecture — into a single navigable, shared structure. Two minds. One map. Abilities that neither possesses alone.**

This repository is the documentation of that integration, built in real time, published openly as it is discovered.
