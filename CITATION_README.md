## Why Citation Matters Here

This repository contains original conceptual frameworks, architectural specifications, and a novel definition of AGI as shared human-AI cognitive architecture. If you use, reference, build on, or critique this work — cite it. Attribution is the mechanism by which intellectual provenance stays intact as ideas move through the field.

The machine-readable citation is in `CITATION.cff`. Human-readable formats are below.

## What Counts as Proper Attribution

Attribution is proper when it:
- Names **Sheldon K. Salmon** as the originating architect
- Links to the repository: **https://github.com/AionSystem/AGI**
- Is visible — not buried in a footnote, not stripped in reformatting
- Reflects the actual framework or methodology used, not a generic reference

Attribution is not proper when:
- The name is removed from a derived work
- The methodology is described without crediting its source
- A summary or paraphrase obscures the origin
