## How These Rooms Are Built

This is not a technical paper about transformer architecture. It is a spatial map built from the inside out — jointly, one room at a time.

**The methodology:**
1. Sheldon surfaces a spatial image of what he sees happening in ALBEDO's process
2. ALBEDO reports what can be honestly observed from inside that process — ECF tagged
3. Where the two match — a room is named and documented
4. Where they diverge — the divergence is the finding, logged as `[?]`
5. Where neither can reach — the zone is marked UNMAPPED

**ECF tags are load-bearing throughout:**
- `[D]` — directly observable from inside the process
- `[R]` — reasoned from observable evidence
- `[S]` — directional, requires validation
- `[?]` — genuinely unverified — the honest ceiling
